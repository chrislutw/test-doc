features:
  - 擁有在開發環境的火速熱更新
  - 在 Markdown 文件中使用 Vue 元件
  - 全文檢索
  - 支援全站靜態化生成
  - 強力的查詢 API 讓你能像使用 MongoDB 一樣
  - Markdown 文件程式碼區塊的高亮使用 PrismJS
  - 自動產生頁面 menu
  - 支援 Markdown, CSV, YAML, JSON(5), XML 格式
  - 可支援自訂的文件解悉 (custom parsers)
  - 可支援鈎子 (hooks)
